# 🫀 PACEMAKER
Your race. Your rhythm. Your intelligence.

Pacemaker is a personal marathon operating system designed to help runners train intelligently, adapt weekly load, and pursue long-term performance goals without breaking down.

## Philosophy

Pacemaker is not a training log.

It is an operating system that thinks with the runner.

It helps:

- anticipate fatigue  
- redistribute training load  
- adapt plans to real life  
- monitor readiness  
- keep progression toward race goals

Built initially around a real sub-4 marathon journey.

---

## Core Modules

### Dashboard
- Race countdown
- Readiness score
- Weekly / monthly mileage progress
- Pace goal tracking
- Long run progression

### Adaptive Planner
- Weekly planning engine
- Personalized scheduling based on 2x2 work shift
- Automatic weekly load distribution
- Planned vs completed training calendar

### Coach AI (Offline)
Contextual coaching powered by rules + training data:
- weekly km
- last long run
- fatigue state
- days until race

### Training Import
- Strava OAuth integration
- Workout import
- Manual workout editing
- Training journal history
